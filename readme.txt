Installation
1. Import node red flow to Node Red (node_red_flow_DTS_1946v2.json) - result: https://prnt.sc/AEc0YP2-Exfp
2. Set up flow server on all (3) modbus flex getters (usb0?) - https://prnt.sc/wDvXNRPDP793
3. Configure the MQTT server (HA) on the MQTT out node (DTS1946_4P) - connection and security tab.
4. Check MQTT Explorer if the data is published: https://prnt.sc/PsACibIjqO43
5. Configure sensors in HA - data from configuration_DTS_1946.yaml
